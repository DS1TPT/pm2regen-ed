pm2regen-ed
============
pm2regen-ed는 프린세스 메이커 2 리제네레이션을 위한 아주 작고 간단한 텍스트 인터페이스(TUI) 세이브 파일 편집기입니다.   
   
pm2regen-ed는 게임 이용자들이 수많은 엔딩을 막대한 시간을 들여 보는 대신 보다 쉽고 빠르게 수집하거나, 게임 진행 중 버그로 인해 스탯이 정상적으로 변화하지 않았을 때 이를 고치고 진행함은 물론, 게임을 특이한 방식으로(예를 들어, 게임을 시작할 때 고작 500 골드 대신 대자본을 가지고 시작하는 것) 즐길 수 있게 하는 등 플레이 경험을 향상시키기 위한 도구입니다.   
pm2regen-ed는 다음 스탯들을 고칠 수 있습니다:
1. 체력, 근력 등 8대 기본 능력치
2. 스트레스와 소지금
3. 4대 평가치
4. 마법기술 등의 평가 능력치
5. 숨겨진 스탯(모성, 왕자/아버지/집사(큐브)와의 관계치, 몬스터 살육 수)
6. 인맥
7. 신체 치수(키, 몸무게, 쓰리 사이즈)
   
이 소프트웨어는 Github의 kokke 님의 tiny-AES-c 를 사용합니다. 다음 주소에서 소스코드와 여러 정보를 보실 수 있습니다: https://github.com/kokke/tiny-AES-c/   
   
요구사항
--------
- POSIX 호환 운영체제(Linux, BSD, macOS ...) 또는 윈도우 10 버전 1511 이상
- ANSI VT 표준을 지원하는 터미널 에뮬레이터(요즘 운영체제는 이미 기본으로 설치돼있음)
   
사용법
------
1. 미리 빌드된 바이너리 파일은 Releases 페이지에서 내려받으실 수 있습니다. 원하시는 기본 언어 설정을 가진 압축파일을 내려받으십시오. 파일의 압축을 해제하신 다음 실행 파일과 런처를 바로 프린세스 메이커 2 리제네레이션의 세이브 폴더에 넣으신 뒤 실행하시면 됩니다.   
2. 실행 파일과 런처를 세이브 파일 디렉토리에 놓습니다. 세이브 파일은 보통 아래의 디렉토리에 있습니다:
```
윈도우: %localappdata%\BrisBrain\PM2\Saved\SaveGames\<숫자>
리눅스: ~/.steam/steam/steamapps/compatdata/2311530/pfx/drive_c/users/steamuser/AppData/Local/BrisBrain/PM2/Saved/SaveGames/<숫자>
```
3. 런처를 더블 클릭하거나(리눅스), 실행 파일을 더블 클릭(윈도우)해 pm2regen-ed를 실행합니다.   
주의: 리눅스에서는 실행 파일 자체를 더블 클릭하여 실행하지 마십시오. 실행 파일을 더블 클릭하는 경우, 실행 파일이 터미널에 연결되지 않은 채 백그라운드에서 작동하게 됩니다. 런처 데스크톱 엔트리로 실행하십시오.
```
윈도우: pm2regen-ed.exe를 더블 클릭하여 실행합니다.   
리눅스: "Launch pm2regen-ed"(데스크탑 엔트리. pm2regen-ed와 같은 디렉토리에 있어야 합니다)를 더블 클릭하여 실행합니다.
```
   
터미널에서 실행하려는 경우 아래와 같이 실행하십시오:
```
리눅스: ./pm2regen-ed <언어>
윈도우: pm2regen-ed <언어> (명령 프롬프트)
        .\pm2regen-ed <언어> (Powershell)

/* 파일 선택기를 사용하지 않을 때 */
리눅스: ./pm2regen-ed <언어> <세이브 파일 번호>
윈도우: pm2regen-ed <언어> <세이브 파일 번호>
```
지원 언어: en(영어), kr(한국어), jp(일본어)   
인자로 언어만 전달되었거나 인자가 없는 경우(프로그램을 더블클릭해 실행한 경우 포함) 파일 선택기가 표시됩니다.
인자로 파일 번호만 전달받은 경우(예시: ./pm2regen-ed 2), pm2regen-ed는 빌드시 기본으로 설정한 언어를 표시하며, 파일 선택기를 건너뛰고 userData<번호>.bin을 불러옵니다.   
pm2regen-ed가 성공적으로 세이브 파일을 불러오면 곧바로 userData<번호>_bak.bin 이라는 백업 파일을 생성합니다. 만일 불러오려는 세이브 파일과 같은 번호를 가진 백업파일이 있다면, 해당 백업 파일은 새로운 세이브의 백업 파일로 덮어씌워집니다. 똑같은 번호를 가진 세이브 파일을 연달아 수정하시려는 경우라면 이 점을 유념하십시오.   
   
참고: <번호>가 1인 경우, pm2regen-ed는 "userData.bin"을 불러옵니다. 프린세스 메이커 2 리제네레이션이 첫째 세이브 파일을 "userData1.bin"이 아닌 "userData.bin"이라 이름을 붙이기 때문으로, 의도된 동작입니다.   
   
빌드하기
---------
윈도우와 리눅스용으로 빌드된 실행 파일을 Releases 페이지에서 내려받으실 수 있습니다.   
만약 직접 pm2regen-ed를 빌드하시려면 아래의 명령어로 이 레포지토리를 내려받으십시오:   
```
git clone --recursive https://github.com/DS1TPT/pm2regen-ed.git
```
모두 내려받으신 다음에, (pm2regen-ed의 루트 디렉토리)/build 디렉토리로 들어가신 다음, CMake로 빌드하십시오. 리눅스에서 빌드하는 예제는 아래와 같습니다:   
```
cmake ..
make
```
CMake 없이도 빌드하실 수 있습니다. pm2regen-ed의 루트 디렉토리로 들어가신 다음 아래의 명령어를 실행하면 GCC로 pm2regen-ed를 빌드할 수 있습니다:
```
gcc -I./inc -I./lib ./src/*.c ./lib/tinyAES/aes.c -o pm2regen-ed
```
이 소프트웨어는 _WIN32 매크로가 정의되지 않은 경우 POSIX 호환 운영체제용 바이너리로 빌드됩니다.   
이 프로그램의 기본 언어를 바꾸시려면 main.h의 27번 줄을 고치십시오.
